# Bangalore-Housing-Price-Prediction
Developed a regression model to predict property price using features like location, total_sqft, bhk, bath, and area_type.
Performed feature engineering (price_per_sqft), location-wise and domain-driven outlier removal, dimensionality reduction, and categorical encoding.
Trained Linear, Ridge, Lasso, and ElasticNet models and evaluated using R² and Cross-Validation, selecting the final model based on stability and generalization.
